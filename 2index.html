BASIC PRINCICIPLES OF WEBSITE DEVELOPMENT

purpose

For every page in the wabsite should have a clear purpose, which fulfill the need for your website user.

well structured

Organise the information using headlines and subheadlines, using bullet points instead of long windy sentences, and cutting the waffle.

typeface

mostly, sans serif font are used to write the content of website as it is easy to read online such as time new Roman, Arial,etc.

Images

inoder to improve ranking of website and to connect with target audience. it is very necessary to use right image for website.
